จงเขียนส่วนของ declaration (หรือ prototype) ในส่วนของการนิยามฟังก์ชั่น จากข้อความต่อไปนี้ โดยให้นิสิตกำหนดชื่อตัวแปรที่จำเป็นต้องใช้เอง
- ฟังก์ชั่นชื่อ sayhi ไม่มีการรับค่า ไม่มีการส่งค่ากลับ
- ฟังก์ชั่นชื่อ triangle รับค่า int 1 ตัว, float 1 ตัว ส่งค่ากลับเป็น float
- ฟังก์ชั่นชื่อ showme รับค่า int 2 ตัว, float 1 ตัว ไม่มีการส่งค่ากลับ
- ฟังก์ชั่นชื่อ calculate ไม่มีการรับค่า ส่งค่ากลับเป็น int
ให้เติมคำตอบลงในช่องว่างตามลำดับ

/*\*Ans1\**/ void sayhi(void);
/*\*Ans2\**/ float triangle(int a,float b);
/*\*Ans3\**/ void showme(int a,int b,float c);
/*\*Ans4\**/ int calculate(void);
