จงเขียนโปรแกรมเพื่อทำการคำนวณหาค่าฟังก์ชั่น cos(x) โดย x มีหน่วยเป็นเรเดียน
โดยใช้สมการต่อไปนี้
\[cos(x)=1-\frac{x^2}{2!}+\frac{x^4}{4!}-\frac{x^6}{6!}+...\]

โดยโปรแกรมจะทำการอ่านค่าของ x สำหรับคำนวณหาค่าฟังก์ชั่น cosine และ n สำหรับจำนวน term
ที่จะนำไปใช้ในการคำนวณจากผู้ใช้ เช่น ถ้าผู้ใช้กำหนดค่า x=0.5 และ n=5 โปรแกรมจะทำการคำนวณดังนี้

\[cos(0.5)=1-\frac{0.5^2}{2!}+\frac{0.5^4}{4!}-\frac{0.5^6}{6!}+\frac{0.5^8}{8!}\]

Sample output 1

Input x : 0.5
Input n : 5
cos(0.50) = 0.878 rad

Sample output 2

Input x : -0.785
Input n : 8
cos(-0.79) = 0.707 rad
